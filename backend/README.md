## Installation

## Local setup

#### Clone the repo

    git clone https://github.com/iTs-rd/Flipr.git
    cd Flipr/backend

#### Install node modules

    pip install -r requirements.txt

#### Running the server

    python manage.py runserver

Visit [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser.
